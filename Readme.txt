Данный проект создан только в учебных целях. Заполненная база для импорта в папке 'sqlDatabase'
Версия 0.1.2.


1) Доступен вход и регистрация на сайте по клику на человечка справа.
2) Отборажаются фильмы и сериалы добавленные в базе данных.
3) При нажатии на фильм и сериал мы поподаем в детальный просмотр информации о фильме(доступен также просмотр через ютюб фильма) или сериале.
4) Есть возможность добавить в избранное фильм или сериал. При добавлении запрос отправляется через ajax и при успехе страница отобразит это и перезагрузится.
5) Можно перейти в профиль, в избранное или выйти.

Функционал поиска в момент написания readme не работает(стоит заглушка).
